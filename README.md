多元升学初探画像卡（AI升学提示语生成器）

本项目是一个面向高中生及其家长的 AI升学路径提示语生成工具，通过填写学生基本信息、学业成绩、身体条件、家庭预算等关键字段，自动生成一段结构化提示语，可一键复制并粘贴至“象导AI”智能体平台（[https://zt-rc.xdsy.cn/pages/index/index](https://zt-rc.xdsy.cn/pages/index/index)），用于获取个性化的多元升学路径推荐（如强基计划、综合评价、专项计划、中外合作办学、艺术体育类、军警公安类等）。

✨ 功能亮点
智能表单联动：
选择高中自动填充学籍所在地（支持“其他”手动输入）
户籍城市与县/区二级联动（针对广东部分地市）
必填校验与错误高亮：确保关键信息完整
多选职业倾向：支持医学、教师、军警公安、飞行员、航海等方向筛选
结构化提示语生成：将用户输入自动拼接为 AI 可理解的自然语言提示
一键复制 + 自动跳转：提升使用效率，无缝对接智能体平台

📝 表单字段说明

字段 类型 说明
------ ------ ------
就读高中 下拉 + 文本 内置4所示范高中，支持“其他”自定义
学籍所在地 自动/手动 根据高中自动填充，若选“其他”则需手动输入
户籍类型 城市+区县联动 支持广东省多地级市及下辖区县（如揭阳→惠来）
性别 / 年级 / 年龄 必选下拉 年龄按周岁分段
成绩定位 文本 可输入“年级100名”、“目标特控线”、“985”等
英语成绩 数字（0–150） 按高考150分制
裸眼视力 下拉 按视力值对应近视度数范围
身高（cm） 数字（140–220） 用于军警、航空等专业筛选
色觉状况 下拉 区分正常/色盲色弱
职业倾向 多选复选框 不勾选则默认不考虑相关方向
是否接受定向就业 必选 影响地方专项、公费师范生等推荐
家庭学费预算 必选 分5档，最高“30万以上”
竞赛奖项 / 艺体特长 可选下拉 支持五大学科竞赛、科创、艺体等类别
💡 所有带 * 的字段为必填项。

🛠 技术实现
纯前端实现：HTML + CSS + JavaScript，无后端依赖
响应式设计：适配手机和平板屏幕
安全复制：优先使用 navigator.clipboard，降级使用 document.execCommand
用户体验优化：
错误自动滚动聚焦
“其他”选项动态显示/隐藏
提示语结果区域清晰展示

🚀 使用方式

1. 在浏览器中打开 index.html
2. 完整填写表单（必填项标红）
3. 点击 【✨ 生成我的AI升学提示语】
4. 点击 【📋 一键复制】 → 
5. 在平台粘贴提示语，即可获得个性化升学路径建议
